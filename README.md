## Instalacion

Frontend dependencies:

    $ npm install

Run the development server:

    $ npm run 

Backend dependencies:

    $ pip install -r "requirements.txt"
    $ cd api 
    $ python ./crear_db.py #creates a database with users
    $ python ./user.py #run api server
